# 基于 UE4 与 Python 的智能无人机路径规划系统

**项目级别**：大学生创新创业训练计划 (大创)
**所属院校**：长安大学
**项目周期**：2024 - 2025
**技术栈**：Unreal Engine 4 (UE4) / Python / 无人机飞控算法

## 项目简介 (Abstract)
本项目针对智能无人机的三维空间路径规划问题，构建了一套基于 UE4 虚拟引擎与 Python 联合仿真的控制系统。通过 UE4 搭建高逼真度的物理仿真场景，并利用 Python 编写核心路径规划算法与飞控逻辑，实现了无人机的自主避障与最优路径寻优。

> **注**：由于 UE4 工程包含海量的高精度三维资产与材质库（体积庞大），本目录仅作为项目成果、学术文档与核心结项材料的归档。完整工程源码与物理资产未包含在内。

## 核心系统架构
1. **UE4 仿真端**：负责物理引擎解算、碰撞检测反馈以及实时渲染高精度环境。
2. **Python 算法端**：作为系统的"大脑"，接收 UE4 传回的环境传感器状态数据，运行相关算法实时计算下一帧的飞控指令（俯仰、偏航、滚转、油门）。
3. **通信机制**：采用 AirSim 实现引擎端与算法端的高频数据交互。
