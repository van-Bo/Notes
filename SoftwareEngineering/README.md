# 🏗️ Software Engineering | 软件工程

本目录主要归档软件开发生命周期 (SDLC) 中的标准文档模板与课程设计资料。

## 📄 文档模板 (Templates)

| 文件名 | 描述 | 阶段 |
| :--- | :--- | :--- |
| **SRS** | **软件需求规格说明书** (Software Requirements Specification) | 需求分析 |
| **HLD** | **概要设计说明书** (High-Level Design) | 架构设计 |
| **LLD** | **详细设计说明书** (Low-Level Design) | 模块设计 |
| **Testing** | **软件测试计划与报告** | 测试验收 |

这些模板遵循国家标准/IEEE标准，适用于规范化的课程设计或项目开发。
